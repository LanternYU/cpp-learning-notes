# 二.关于指针
# 2.1 指针实例
```cpp
#include<iostream>
using namespace std;
int main(){
	int a=10;//声明了一个 int 型变量a a的值为10 假设a的地址为0x1234
	int* p;//声明了一个 指向int型的指针 p
	p=&a;//把a的地址赋给p
	cout<<p<<endl;//输出a的地址 0x1234
	cout<<*p<<endl;//输出10
}
```
# 2.2 指针的本质：地址
对于上一段代码的最后一句
```cpp
cout<<*p<<endl;
```
程序执行逻辑如下：  
第一步：访问p的地址  
第二部：读取p的地址中的内容（值） 发现为a的地址  
第三步：访问a的地址  
第四步：读取输出a的地址中的内容 即a的值

那么指针的大小是多少个字节 和指向的数据类型有关吗？  
**无关** 所有指针类型的大小相同，因为指针本质是「内存地址」，**大小取决于系统的地址总线位数**：  
地址总线为 32 位，指针大小为 4 字节（可表示 0 ~ 2^32-1 的地址空间）。  
地址总线为 64 位，指针大小为 8 字节（可表示 0 ~ 2^64-1 的地址空间）。  
因此，int* 和 double* 的大小完全一致（与指向的类型无关），仅由系统位数决定。
# 2.3 明晰概念：*与p 
#是一个运算符 被称为间接值或者接触引用  
它被用于与指针 可以得到该地址存储的值  
假设p是一个指针 则p表示的是个地址 而*p则表示改存储在地址的值
